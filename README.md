# ゲーム のタイトル

* 恐竜ゲーム

## 実行環境の必要条件

* python 3.10
* pygame >= 2.1

## ゲームの概要

Googleの恐竜ゲームのpygame版です。ジャンプやしゃがみで障害物を避けるゲームです。

## ゲームの実装

### 共通基本機能

* 恐竜が走る
* ジャンプする
* しゃがむ

### 担当追加機能

* スコアの追加
* 障害物の追加:障害物として鳥を呼び出した。画像を二つ使い、飛んでいるようにしている。
* 障害物の追加2:大小のサボテンを追加した。
* 背景装飾

- 雲を背景に描画
-　ステージの移動に合わせてうごくようにする

* サウンドの追加
